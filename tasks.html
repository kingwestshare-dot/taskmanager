Connect to Google Drive

Enter your Google OAuth Client ID to sync tasks across all your devices. Your tasks are saved as a private file only this app can access. How to get a Client ID →

Getting your Client ID (takes ~10 min, free):

1. Go to console.cloud.google.com and create a new project
2. Go to APIs & Services → Library, search Google Drive API, click Enable
3. Go to APIs & Services → OAuth consent screen → External → fill in app name → Save & Continue through all steps
4. Go to APIs & Services → Credentials → Create Credentials → OAuth client ID
5. Application type: Web application
6. Under Authorized JavaScript origins add: null (for local files) or your hosted URL
7. Click Create — copy the Client ID and paste it above
